## 제공할 서비스
- 사용자 간의 블로그 게시판 기능 제공
- 다양한 주제의 게시글 작성 및 검색
- 사용자 피드백을 반영한 댓글 작성 기능
## 주요 기능
### 기본기능
* 회원가입 및 로그인 기능
* 블로그 글 작성 / 읽기 기능
* 글 수정 / 삭제 기능
* 게시글의 제목 및 내용을 기반 검색 기능 
### 추가기능
* 댓글 달기 기능
* 게시글 카테고리별 분류 기능 ex) 영화 리뷰 , 일상 , 후기 등
